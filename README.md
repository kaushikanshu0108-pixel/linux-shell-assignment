---

## Scripts Included in This Repository

### 1. **backup.sh**
This script is used to create a backup of any file.  
It asks the user to enter the filename, creates a backup folder with the current date, and stores the copy of the file inside that folder.  
This helps to automate regular backups and avoid data loss.

---

### 2. **system_check.sh**
This script displays important system information such as:
- Current date and time  
- System uptime  
- Top CPU-consuming processes  
- Memory usage  
- Disk usage  

This script helps in monitoring the overall performance and health of the system.

---

### 3. **netcheck.sh**
This script checks network connectivity and provides the following details:
- Internet connection status (using ping)
- System IP address  
- Default gateway information  
- DNS server details  

It is useful for basic network troubleshooting.
